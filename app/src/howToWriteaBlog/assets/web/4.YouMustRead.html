1. Affiliate

An organization or individual whom you have formed an official or informal agreement with either sell or receive products, services, content, traffic, etc.

2. Alexa Rank

The Alexa Rank is a ranking system, this ranking is making available by Alexa.com. Which provides rank on traffic levels for websites? The Alexa rank is deliberate according to the quantity of users who have visited a website by means of the Alexa toolbar installed.

3. ASP (Active Server Pages)

ASP is a server side scripting language formed by Microsoft Corporation Limited, for dynamically and interactively created web pages and database functions. The default scripting language used for writing ASP is VBScript; Web pages created with ASP usually have an .asp suffix.

4. Auto responder

An email message that is sent mechanically in response to a message to a different email address.

5. Backlinks

Backlinks are also known as incoming links, inbound links, inlinks, and inward links to a website or web page. In the search engine optimization (SEO) world, the number of backlinks is one sign of the reputation or significance of that website or page. If your contents are high quality, you can get many quality backlinks. If the website that is receiving backlinks gets many High Quality and High Page Rank, the possibility is high position on the Search Engines.

6. Bandwidth

It is a rate of data transfer. A word Bandwidth used to explain how much data you could send through a connection to the net, measured in bits per second (bps). How much data can transmit in a given amount of time? The higher bandwidth rate means faster the rate of data transmission.

7. Banner Advertising

The Long thin ads usually located at the top and bottom of web pages. Selling advertising space in the form of banner ads on your site is a good way to earn fair money online; if your site offers information is would be useful to surfers and they will be more willing to accept seeing the ads from which you derive income.

8. Baud

A French engineer, Jean-Maurice-Emile Baudot first introduced it. It was first used to measure the speed of telegraph transmissions. This is the speed at which modem transfer data. One baud is roughly equal to one bit per second. It takes eight bits to make up one character. For example, 400 baud means that 400 bits are transmitted each second. (400 bps).

9. Blog Publishing Software

BPS is a software designed to make simpler the formation and preservation of web blogs. This application support the authoring, control, editing, and publishing of blog posts and comments.

10. Bookmark

A roll of links to URLs on the Internet maintained within a web browser. All modern web browsers include bookmark features. Bookmarks are usually access through a menu in the user's web browser.

11. Browser

A client program or a software application that is used to retrieving, locate, display, and presenting resources on World Wide Web. The two most popular browsers are Microsoft Internet Explorer and Mozilla Firefox.

12. Cookie

It is a small piece of text stored on a user's computer. A file stored on your hard disk and used to recognize your computer or your preference to a remote computer. Cookies are frequently used to recognize visitors to web sites. Cookies contain information that can help speed access on subsequent visits, such as passwords and details of the user's display facilities.

13. CPC (Cost per click)

CPC is the amount of money an advertiser pays for search engines and other Internet publishers for a single click on its advertisement that brings one visitor to its website. (CPC) vary depending on the search engine and the level of competition for a particular keyword.

14. CPM / CPI (Cost per thousand impressions)

It is an expression frequently used in online advertising and marketing related to web traffic. It is used for measure the worth and cost of a specific e-marketing campaign. The CPM model refers to advertising buy base on impression. Each time an advertisement loads onto a user's screen, the ad server may calculate that loading as one impression.

15. Domain
Domain is a set of network addresses. The smaller network making up the Internet, and having a lot of computers or even smaller network within it, is called a domain. The domain may represent either a type of organization or a geographical location.

16. Domain Name

An addressing scheme assigns names and numbers to identify the computers on the Internet based on the Domain Name System (DNS). An imperative purpose of domain name is to give easily identifiable and memorial names to numerically addressed Internet resources.

Originally created by Macromedia, and is now developed by Adobe Systems, which acquired Macromedia in 2005. Latest versions have incorporated support for web technologies such as CSS, JavaScript, and various server-side scripting languages and frameworks including ASP and PHP.

18. Facebook

Facebook is a popular free social networking website that allow registered users to make profiles, share links, upload photos and video, send messages and stay in touch with friends and family. Facebook offer a choice of solitude option to its members. A member can make all his communications able to be seen to everybody, he can chunk specific connections, or he can stay all his communications confidential.

19. Firewall

Software intended to put off unauthorized right of entry to a computer network. Firewalls can be implemented in either hardware or software, or a combination of both. All messages toward the inside or send-off the intranet pass through the firewall, which examine each message and block those that do not meet the particular safety criterion.

20. Flash

It is popular authoring software developed by Macromedia. Flash has become a popular system for adding animation and interactivity to web pages. It is more often than not used to make animation, advertisements, and a variety of web page components, to put together video into web pages, and more recently, to expand rich Internet applications.

21. Forums

An online discussion group, which participants with common interests can exchange. It is such a site where people engage themselves in discussion of common interest subjects, ask questions, reply questions, and create a social atmosphere online.

22. FTP

File Transfer Protocol. A specific set of protocols used by computers connected over the Internet to transfers files. It allows you to move files between two computers on the Internet. To move or transfer files with FTP, you make use of a program often call the client. The FTP client program initiates a link to a remote computer running FTP "server" software. After the connection is recognized, the client can decide to drive and get copies of files, individually or in-group. To connect to an FTP server, a client requires a username and password as set by the administrator of the server. Many public FTP archives go after a particular convention for that accepts a username of unnamed.

23. HTML

Hypertext Markup Language is the encoding scheme used for creating a Web document. The computer that maintains the document (host computer), and the computer that display the document may be at different geographical locations. It can be used on any platform or desktop. An HTML document consists of text and tags for conveying the data of a document and to mark its structure. The tags in an HTML document are referred to as element.

24. HTTP

Hyper Text Transport Protocol is the set of rules for transfer information and files such as graphics, images, sound, video, and other multimedia files on the World Wide Web. It defines how messages are made format and transmit, and what procedure Web servers and browsers should get in response to various commands.

25. IP (Internet Protocol)

Internet Protocol address is the numeric address used to locate physical or virtual computers or sites on the Internet. The Internet Protocol (IP) is the system or protocol by which data is sent from one computer to another on the Internet. Each computer (known as a host) on the Internet has at least one IP address that uniquely identify it from all other computers on the Internet

26. JPEG

27. LAN

Local Area Network, A computer network or a private transmission with certain limitations like networking within same building or campus for sharing voice, data, and video.

Search engines to index a page so that somebody searching for the kind of information the page contain will be able to find out information that you give in a Meta tag. There are quite a lot of kinds of Meta tags, but the most significant for search engine indexing are the keywords Meta tag and the description Meta tag.

41. RSS

RSS (Really Simple Syndication) is an XML-based format. They advantage readers who wish to subscribe to timely updates from preferential websites or to aggregate feeds from many sites into one place. . An RSS document, which is called a "feed," can be read using software called an RSS reader or feed reader or aggregator. A Web site that wishes to allow other sites to publish some of its content creates an RSS document and registers the document with an RSS publisher. A user that can read RSS-distributed content can use the content on a different site.

A software application or service used to locate files on an internet or the Web. Normally access with browsers, a few of the most common web search engines includes Google, Yahoo, and WebCrawler. It consists of a string that supplies the Internet address of a resource on the WWW along with the protocol by which the resource is accessed.
